# Task Manager

A simple **task management** web application built with React and Vite.  
Live demo: https://taskmanager-lilac-eta.vercel.app/

## Features

- Add new tasks with a title.
- Mark tasks as completed or pending.
- Delete tasks from the list.
- Responsive and clean user interface.

## Tech Stack

- React (Vite)
- JavaScript (ES6+)
- HTML, CSS
